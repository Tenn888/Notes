# Notes — настольные заметки на Tkinter

Минималистичное приложение для быстрых текстовых заметок на Python. Без БД и лишних зависимостей — все файлы сохраняются рядом с программой в виде `notes_#.txt`.

## Возможности
- Создание новой заметки в отдельном окне.
- Автоматическая нумерация и сохранение заметок в UTF-8.
- Просмотр всех заметок в прокручиваемом списке.
- Редактирование существующей заметки одним кликом.
- Удаление заметки с последующей пере-нумерацией файлов.

## Установка
1) Убедитесь, что установлен Python 3.10+.
2) Установите зависимости:
```bash
pip install tkinter TKinterModernThemes os
```

## Запуск
```bash
python main.py
```
После старта откроется окно с существующими заметками. Кнопка «+» в правом нижнем углу создает новую.

## Горячие детали реализации
- Все заметки хранятся в текущей директории и имеют формат `notes_N.txt`, где `N` — целое число без пропусков.
- Интерфейс построен на `tkinter` с использованием темы TKinterModernThemes.
- Список заметок оборачивается в `Canvas` с полосой прокрутки — удобно при большом количестве файлов.

## Подсказки по использованию
- Чтобы быстро открыть заметку на редактирование, кликните по ее содержимому в списке.
- При удалении заметки последующие файлы сдвигаются, чтобы нумерация оставалась плотной.
